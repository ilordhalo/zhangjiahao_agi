# PRD

## 产品目标

把 ZHA-5 的 AGI 调研报告转译成一个可本地打开的中文交互式网页，让读者快速理解“哪些当前 AI 进展和 AGI 更相关”，并能继续追溯到完整研究报告与来源。

## 目标用户

- 想快速浏览 AGI 相关进展优先级的个人研究者。
- 需要向他人解释 AI/AGI 进展结构的产品、技术或投资分析人员。
- 希望在本地无后端环境下演示调研结论的项目维护者。

## 核心范围

- 首屏以“AGI 进展观测台”为主题，直接呈现结论，不做营销式空页面。
- 展示 AGI 相关性排序，突出长周期智能体、推理时扩展、世界模型与具身智能。
- 提供筛选式能力图谱，按智能体、世界与具身、评测安全、基础设施分组探索。
- 展示未来 6-12 个月观察指标和反信号，避免把单点演示误读为稳定 AGI。
- 保留完整 ZHA-5 研究报告作为可追溯来源。

## 非目标

- 不建设后端、CMS、用户登录或在线数据抓取。
- 不声明任何公司已经实现 AGI。
- 不自动更新外部来源；本版本以 2026-07-07 的 ZHA-5 报告为内容基线。

## 验收标准

- `index.html` 可直接打开，或通过 `python3 -m http.server` 本地预览。
- 桌面和移动端布局无文本重叠、无空白主区域。
- 交互筛选按钮可更新卡片内容和 `aria-live` 状态。
- 动画具备 `prefers-reduced-motion` 降级。
- `python3 scripts/validate_site.py` 通过。
