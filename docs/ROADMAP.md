# Roadmap

## 已交付

- 本地静态 AGI 信息网站。
- 基于 ZHA-5 报告的中文内容结构。
- 能力方向筛选、渐入动画、canvas 首屏可视化和 reduced-motion 支持。
- PRD、Architecture、Roadmap 与静态验证脚本。

## 后续候选方向

- 增加“证据强度”视图，把强证据、中强证据、中证据单独筛选。
- 增加来源详情抽屉，让每张能力卡直连报告中的代表性来源。
- 增加打印或导出模式，方便离线汇报。
- 在未来调研更新后，把 `domainData` 从手写数组迁移为由报告元数据生成。

## 约束

- 在没有明确需求前保持静态部署，不引入框架、构建链或后端。
- 内容更新必须保留日期范围，避免把历史报告误当作实时资讯。
- 动画不能降低可读性，移动端优先保证稳定布局和触控可用性。
